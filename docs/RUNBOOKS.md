# Runbooks

- `runbooks/closeout.md` — checks before declaring work complete.
- `runbooks/openai-tunnel.md` — foreground OpenAI Secure MCP Tunnel setup for the local Obsidian MCP server.
- `runbooks/local-release.md` — deploy a clean commit to the user-level gateway binary, verify the tunnel, or update from `origin/main` and release.

Future runbooks should stay procedural. Architecture and requirements belong in their owning docs.
